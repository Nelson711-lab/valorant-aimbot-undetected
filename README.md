# Valorant Aimbot v3.1.0

External aim assist for Valorant. Undetected as of May 2026. Works in competitive and unrated modes.

## Features
- Smooth aimbot with humanized movement
- Configurable FOV and smoothness
- Triggerbot with reaction time randomization
- Color-based detection (no memory reading)
- Vanguard-safe (external only, no kernel access)

## System Requirements
- Windows 10/11 64-bit
- Python 3.11+
- Valorant installed and updated
- 1920x1080 resolution recommended

## Quick Start
1. Extract all files to a folder
2. Run `install.bat` as Administrator
3. Launch Valorant and enter a match
4. Run `python main.py`
5. Press F2 to toggle aimbot, F3 for triggerbot

## Keybinds
| Key | Function |
|-----|----------|
| F2 | Toggle Aimbot |
| F3 | Toggle Triggerbot |
| F6 | Cycle target (head/chest) |
| END | Emergency exit |
